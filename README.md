Student Management System

Description

A Python-based console application to manage student records efficiently.

Features

- Add Student
- View Students
- Search Student by Roll Number
- Search Student by Name
- Update Student Details
- Delete Student
- Count Total Students
- Data Storage using File Handling

Technologies Used

- Python
- File Handling
- Git
- GitHub

Project Structure

StudentManagementSystem/

- main.py
- students.txt
- README.md

How to Run

1. Open the project folder in VS Code.
2. Open Terminal.
3. Run:

python main.py
